# playwright_assignment_bs1572

Playwright + TypeScript automation project with POM architecture, CSV-driven tests, HTML reporting, and multi-browser execution support.

## Features

- **Automation Testing**: Comprehensive test coverage using Playwright.
- **CSV-Driven Tests**: Dynamic test data management through CSV files.
- **HTML Reporting**: Detailed test reports generated in HTML format.
- **Multi-Browser Support**: Execute tests across different browsers seamlessly.

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/playwright_assignment_bs1572.git
   ```

2. Navigate to the project directory:
   ```bash
   cd playwright_assignment_bs1572
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Run the tests:
   ```bash
   npx playwright test --reporter=html --workers=1 --project=chromium
   ```
